# glTFRuntimeAlembic

glTFRuntime support for Alembic assets.

Requires:

https://github.com/rdeioris/glTFRuntime

https://github.com/rdeioris/glTFRuntimeGeometryCache

Supports:

* Static Meshes
* Curves (for groom/hair or spline)
* Geometry Caches
